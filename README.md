# Achei! - Sistema de Achados e Perdidos

O **Achei!** é uma plataforma web desenvolvida como projeto final para a disciplina de Engenharia de Software (GCC 188). O objetivo do sistema é centralizar o registro de itens perdidos e encontrados em uma comunidade (como um campus universitário, por exemplo), gerenciando de forma eficiente o fluxo desde o recolhimento do objeto até a sua devolução segura ao dono.
